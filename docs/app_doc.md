## Splash (обязательный)

Назначение:
- проверка авторизации;
- инициализация Firebase;
- редирект:
  o  авторизован → Главная;
  o  нет → Авторизация.
________________________________________
2. Авторизация
- вход / регистрация;
- email / телефон (достаточно одного варианта на старте);
- переход в приложение.
________________________________________
3. Главная (основной экран)
   Лучше сразу сделать список заведений (без карты):
- список ресторанов;
- название;
- рейтинг;
- кнопка перехода в карточку;
- кнопка “Оставить отзыв” (можно глобально).
________________________________________
4. Карточка заведения
- название;
- рейтинг;
- базовая инфа;
- кнопка “Оставить отзыв”.
________________________________________
5. Создание отзыва
- 4 критерия:
  o  чистота;
  o  скорость;
  o  еда;
  o  персонал;
- комментарий (опционально);
- кнопка “Далее”.
________________________________________
6. Загрузка чека (ключевой экран)
- камера / галерея;
- превью;
- отправка;
- обработка:
  o  чек уже использован;
  o  ошибка загрузки.
________________________________________
7. Результат
- успех:
  o  “Отзыв принят”;
  o  начисленные бонусы;
- ошибка:
  o  причина отказа.
________________________________________
8. Профиль (минимальный)
- имя;
- бонусный баланс;
- выход.
